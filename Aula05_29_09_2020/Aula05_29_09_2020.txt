Objetivos:

Desenvolver uma aplicação console em C# que leia dois números, some esses dois números e exiba na tela o resultado desta soma.

1. Lógica;
2. Lógica de programação;
3. Sequência lógica;
4. Comandos de entrada, processamento e saída;
5. Variáveis (nomemclatura) e constantes;
6. Tipos.

Antes de codificarmos o problema acima:
1. Entender o problema;
2. Identificar a(s) Entrada(s) - leia, ler, obter, receba, receber, read, etc;
3. Identificar o(s) Processamento(s) - cálculos, etc;
4. Identificar a(s) Saída(s) - escreva, exiba, exibir, mostre, print, etc.

Dispositivos - Sistema de Processamento de Dados:
Entrada(s)      : Teclado, Mouse
Processamento(s): Computador (Processador - CPU, Memória RAM, HD, placa mãe)
                  Sistema Operacional - Windows 7, 8, 8.1, 10/Linux / MacOS
                  Sistema/Aplicativo
Saída(s)        : Display/Monitor/Tela/Impressora

Desenvolver uma aplicação console em C# que |leia dois números|, |some esses dois números| e |exiba| na tela o resultado desta |soma|.

Entrada(s)	Processamento(s)		Saída(s)
Input						Output
n1 10		soma = n1 + n2			soma
n2 20

num1		soma = num1 + num2
num2

numero1		soma = numero1 + numero2
numero2
